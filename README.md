## Development

Start the watch-compile

```sh
npm run res:start
```

Build webpack every time you run

```sh
npm run build
```

## Build from the source code

```sh
npx build
```

## Naming conventions

- `Xyz.res`: a React component
- `xyz.res`: a module
- `webpack.config.js`: the webpack configuration
